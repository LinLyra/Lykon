# 篮球动作标签体系

## V1 必做标签

- idle：静止/等待
- dribble：运球
- pass：传球
- shot：投篮
- jump：跳跃
- sprint：冲刺/突破

## V2 扩展标签

- layup：上篮
- catch：接球
- rebound：抢篮板
- defense_slide：防守滑步
- collision：对抗碰撞
- stop：急停
- crossover：变向

## V3 细分标签

- set_shot：定点投篮
- jump_shot：跳投
- pull_up：急停跳投
- catch_and_shoot：接球投篮
- chest_pass：胸前传球
- bounce_pass：击地传球
- overhead_pass：头上传球
- low_dribble：低运球
- high_dribble：高运球
- behind_back：背后运球
- between_legs：胯下运球
- spin_move：转身
- left_hand_layup：左手上篮
- right_hand_layup：右手上篮
- block：盖帽
- steal：抢断
- box_out：卡位

## 当前不能承诺的标签

- 左脚/右脚起跳
- 完整骨骼姿态
- 真实球轨迹

这些需要视频骨骼、鞋垫传感器或更多身体传感器。
