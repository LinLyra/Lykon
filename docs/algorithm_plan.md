# 算法路线

## 阶段 1：规则模型

目的：快速做 Demo，验证动作信号是否可见。

输入：窗口特征。

方法：峰值、频率、持续时间、左右手差异、UWB速度。

## 阶段 2：传统机器学习

目的：建立稳定 baseline。

模型：RandomForest / XGBoost / SVM / KNN。

输入：手工特征。

## 阶段 3：深度学习

目的：数据量起来后提高动作识别准确率。

模型：tsai / InceptionTime / ResNet / TCN / Transformer。

输入：原始窗口序列。

## 阶段 4：事件引擎

动作不是终点。产品需要篮球事件。

例子：

- pass + catch + 距离合理 => 传球事件
- shot 后多人 jump => 篮板事件
- dribble + UWB朝篮筐移动 => 突破事件

## 阶段 5：视频骨骼校正

视频骨骼不是产品必需输入，而是训练老师。

它用于校正：

- 投篮出手点
- 手臂角度
- 上篮脚步
- 防守姿态
- 左右手出手
